Instruction to Run the Code
===========================

python3 gengen.py {startN} {endN} {step_size} {test_time}
startN is the value of N to start the test
endN is the value of N upto which to run the test
step_size is the value of N to be changed after each test
test_time is the time to run for each test in seconds

The code automatically starts the server from a.out and runs
it on port 8080
